the code is written with nodejs and expressjs with mysql database 

1.Create an endpoint to create a new employee ==> POST /api/add_employee/
2.get list of employees ==> GET /api/get_employees/
3.get list of employees filtered by date ==> GET /api/get_employees/:date/
4.checkin POST /api/check_in/:id
5.checkout PUT '/api/check_out/:id
